# ODSports - Custom Content Management System (CMS)

A specialized and fully tailored Content Management System (CMS) built to streamline operations for sports-centric portals. This platform features a centralized administration panel that gives non-technical users absolute control over dynamic content.

## 🚀 Features
- **Centralized Admin Dashboard:** Comprehensive interface enabling administrators to control every aspect of the portal seamlessly.
- **Dynamic Content & Media Management:** Easily create, edit, update, or remove blogs, media assets, images, and visual content.
- **Event Tracker:** Dedicated modules to manage schedules, matches, tournaments, and event-related information dynamically.
- **Secure Backend Layout:** Follows strict MVC architecture with protected routing and optimized database queries.

## 🛠️ Tech Stack
- **Backend Framework:** Laravel (PHP)
- **Database:** MySQL (Structured Relations & Queries)
- **Frontend Assets:** Blade Templates, JavaScript (ES6+), Bootstrap, CSS3
- **Tools:** Git, Postman for API testing
