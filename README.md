# ODSports CMS

A custom Content Management System (CMS) developed for a sports event platform, enabling administrators to efficiently manage website content, blogs, events, and media through a centralized dashboard.

---

## Project Overview

ODSports CMS was developed to simplify website management by allowing administrators to update dynamic content without modifying the source code. The system provides an intuitive administration panel for managing different sections of the platform while keeping the public website up to date.

---

## Features

- Secure administrator dashboard
- Blog management
- Event management
- Website content management
- Image and media management
- Dynamic content updates
- MySQL database integration
- Responsive administration interface

One of the primary challenges was designing a content management system that allowed non-technical users to update website content easily while maintaining a clean backend architecture and database consistency.
